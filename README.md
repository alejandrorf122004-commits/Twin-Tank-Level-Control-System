# Twin Tank Level Control System

## Overview

This project presents the design, modeling, simulation and control of a coupled twin-tank level system developed as part of a Mechatronics Engineering project.

The system integrates:

- Mechanical design in SolidWorks
- Mathematical modeling
- Transfer function identification
- PID control
- State observer design
- Servo-system implementation
- Python GUI for simulation and analysis
- PLC implementation
- Industrial instrumentation
