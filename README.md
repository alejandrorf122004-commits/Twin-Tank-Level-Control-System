# Twin Tank Level Control System

Second-order coupled tank control system with MATLAB, Python GUI, PLC-oriented implementation, observer design, and servosystem control.

## Overview

This project presents the design, modeling, simulation, identification, and control of a coupled twin-tank liquid level system developed as part of a Mechatronics Engineering project.

The system integrates mechanical design, hydraulic modeling, industrial instrumentation, control theory, graphical simulation tools, and PLC-oriented implementation. The main objective is to regulate the liquid level of the second tank using classical and modern control strategies.

The project includes SolidWorks CAD files, MATLAB scripts, Python GUI tools, experimental data analysis, controller design, state observer design, servosystem equations, and technical documentation.

## Project Objectives

The main objectives of this project are:

* Design and document a physical twin-tank level control system.
* Develop the mathematical model of the coupled hydraulic process.
* Analyze the system as a second-order dynamic process.
* Identify the system behavior using experimental data.
* Design and compare control strategies for liquid level regulation.
* Implement PID-based control approaches.
* Design a state observer for internal level estimation.
* Develop a servosystem structure for reference tracking.
* Create a Python graphical interface for simulation and analysis.
* Prepare the control structure for PLC implementation using TIA Portal.

## System Description

The system consists of two acrylic tanks hydraulically connected through PVC pipes. The input flow is generated by a DC pump controlled through PWM. The main measured variable is the liquid level, obtained using a 4–20 mA industrial level transmitter.

The main controlled variable is the liquid level of the second tank.

The coupled tank system is useful for studying:

* Process control.
* Second-order dynamic systems.
* Mathematical modeling.
* Experimental identification.
* PID control.
* State observers.
* PLC-based automation.
* Graphical simulation and analysis.

## Mathematical Model

The coupled tank system is modeled as a second-order hydraulic process. For two identical tanks, the transfer function from the input flow to the liquid level of the second tank can be represented as:

```text
G(s) = H2(s) / Qin(s)
     = 1 / (A²R1s² + A(2 + R1/R2)s + 1/R2)
```

Where:

* `H2(s)` is the liquid level of the second tank.
* `Qin(s)` is the input flow generated by the pump.
* `A` is the cross-sectional area of the tanks.
* `R1` is the hydraulic resistance between tank 1 and tank 2.
* `R2` is the hydraulic resistance at the outlet of tank 2.

This model was used as the basis for simulation, controller design, observer design, and PLC-oriented implementation.

## Control Strategies

The project includes the design and analysis of different control strategies:

The control objective is to regulate the liquid level of the second tank and achieve stable reference tracking within the operating range of the physical system.

## State Observer

A state observer was designed to estimate the internal states of the system. Since the measured output is the level of the second tank, the observer estimates the behavior of both tanks using the measured output error.

The estimated states are:

```text
xhat1 = estimated level of tank 1
xhat2 = estimated level of tank 2
```

The observer structure improves the analysis of the system and prepares the project for more advanced control strategies.

## Servosystem Implementation

A discrete servosystem structure was prepared for PLC-oriented implementation. The structure uses state estimation, feedback control, and reference tracking.

```text
yhat := xhat2
e_y := y - yhat

u := -0.0052459930*xhat1
     -0.0198923520*xhat2
     +0.0298638303*r

xhat1_n := 0.9938496346*xhat1
           +0.0061503654*xhat2
           +4.4444444444*u
           +0.1593845887*e_y

xhat2_n := 0.0061503654*xhat1
           +0.9894658899*xhat2
           +0.1167913290*e_y
```

Where:

* `r` is the reference level.
* `y` is the measured level of tank 2.
* `yhat` is the estimated output.
* `e_y` is the observer correction error.
* `u` is the control signal.
* `xhat1` is the estimated level of tank 1.
* `xhat2` is the estimated level of tank 2.
